Research

I've been working in several projects as head of NIC Labs (@niclabs), actually I'm focused on:

  • Real-time analytics for DNS packets (project name Rata-DNS).
  • Chilean Internet QoS (project name Adkintun).
  • Threshold Cryptography based virtual HSM (project name Poor's men HSM), this project is partially funded by FRIDA.
  • Robustness of Chilean Internet, project funded by CORFO.
A complete list of my publications can be found in Google Scholar.
